Network Attack Simulator
=======================

Requirements:
- Windows 64-bit operating system
- Administrator privileges
- Npcap installed (https://npcap.com/#download)

Installation Instructions:
1. Extract all files to a folder
2. Run VC_redist.x64.exe and complete the installation
3. Run Npcap-1.80.exe and complete the installation
4. Run dorothy.exe

Note: The application requires administrator privileges to run properly.

Troubleshooting:
- If the application doesn't start, ensure Java 17 is installed
- Verify Npcap is properly installed
- Run as Administrator
